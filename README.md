# Rug-Pull Defender: A Multi-Modal Detection Framework

Official implementation of the paper: **"Rug-Pull Defender: A Multi-Modal Rug-Pull Detection Framework with Transformer Semantics, Static Analysis, and Knowledge-Based Heuristics"**.

## 🚀 Overview
Rug-Pull Defender is an ensemble learning framework designed to identify systemic risks in DeFi. It combines:
1. **Semantic Layer:** Transformer-based modeling of smart contract source code.
2. **Structural Layer:** Static analysis focusing on control flow and vulnerabilities.
3. **Heuristic Layer:** Domain-expert rules and knowledge-based features.

Our model achieves an **F1-score of 0.774** and an **AUC of 0.79** on a manually validated, leakage-proof dataset of 1000 projects.
